アプリを起動する

Unityを起動する

Macにログインし、下のバ」を選択し、「アプリケーション」→「Unity Hub」を起動します。

アプリを起動する

「Unity Hub」が起動したら「プロジェクト」→「WiifitInteractive」と選択してください。

「自動太鼓演奏車」のUnityプロジェクト名は「WiifitInteractive」です。

i

Unityで実行する

選択したプロジェクトが起動したら、▲を押すことでプレイ画面が実行します。

また、以下のWiiFit Balance Boardとの接続については左側のWiiを選択し「Find」ボタンを押して「Found」と表示されれば、WiiFit Balance Boardと接続できていますので、そのままUnityを実行してもつながったままで起動させることができます。(WiiFitとつなげる時もこの手順を実行してください)

※基本的にWiiFit Balance Boardとつなぎっぱなしの状態にしてください

WiiFit Balance Boardの取扱い

WiiFit Balance Board面

WiiFit Balance BoardがBluetoothでしっかりつながっているときはライトが青く光っています

電池を交換する時

WiiFit Balance Boardを裏返してください。

以下の部分の蓋を外してください

電池の蓋を外したら単3電池 4個を入れ替えて交換してください

WiiFit Balance BoardとPCとをつなげる

何らかの原因で接続が切れてしまった場合、PCと再接続する必要があります。

WiiFit Balance Boardのライトが消えていれば、接続が切れている状態です。

WiiFit Balance BoardとPCと接続するにはWiiFit Balance Boardを裏返して赤い●を押すと繋げられる状態となります

WiiFit Balance Boardと繋げられる状態となったら上記UnityでのWiiの「Find」ボタンを押すことでつなげることができます。

WiiFit Balance Boardうまくつながらない

UnityにてWiiの「Find」を押してもerrorになる

WiiFit Balance Boardが見つからない場合

  1. 赤い●を押して、UnityにてWiiの「Find」を繰り返す
  2. Bluetoohと接続されているWiiFitとの接続状況を解除(削除)する
  3. Unityを再起動する
  4. WiiFit Balance Boardの電池を外して入れるを繰り返す

といったことを繰り返してください。

MacでBluetoohとの接続を解除するときは「システム環境設定」→「Bluetooth」を選択するか、ヘッダーのBluetoothのアイコンを選択してください

以下のBluetoohのデバイスの一覧にWiiFit Balance Boardの情報が表示されます。

表示されたWiiFit Balance Boardを選択し、削除してください。

自動太鼓演奏者のソースコードを更新する

バグの修正などを反映する場合の方法について紹介します。(Gitとターミナルを使用します)

Macよりターミナルを起動し、

cd ~/Desktop/projects/WiifitInteractive/

でプロジェクトに移動します。(上記の場所はプロジェクトが置いてある場所になります)

プロジェクトを移動したら、以下のコマンドでGithubより変更内容を取得します

git pull

※ソースコードを更新するためにはWIfiに接続している必要があります。

ソースコードを更新したら、Unity側で自動的に更新がかかり、最新のものに反映されます。
